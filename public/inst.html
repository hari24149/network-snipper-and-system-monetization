Application Setup Instructions

Node.js & Flask App

Setup Instructions

Step 1: Download the Node.js Application

  • Ensure you have Node.js installed. You can download it from Node.js official website.
  • Ensure you have Git installed. You can download it from Git official website.
  • Clone the Node.js application repository from GitHub: 
    git clone https://github.com/hari24149/network-snipper-and-system-monetization
  • Navigate to the cloned directory: 
    cd node-app
  • Install the required dependencies: 
    npm install
  • Start the Node.js application: 
    npm start

Step 2: Download and Setup the Flask Application

  • Ensure you have Python installed. You can download it from Python official website.
  • Download the Flask application from the following link: Download Flask App
  • Extract the downloaded zip file to a directory of your choice.
  • Open a Command Prompt as Administrator:
    • Search for "cmd" in the Windows Start Menu.
    • Right-click on "Command Prompt" and select "Run as administrator".
  • Navigate to the extracted directory in the Command Prompt: 
    cd path\to\your\flask-app
  • Create a virtual environment and activate it: 
    python -m venv venv
venv\Scripts\activate
  • Install the required dependencies: 
    pip install -r requirements.txt
  • Run the Flask application: 
    python app.py
Note: Running the command prompt as an administrator is essential for certain operations that require elevated permissions.

Accessing the Applications

Once the Node.js application is running, open your browser and go to http://localhost:3000 to access it. From there, you can start the Flask application and view its interface.
